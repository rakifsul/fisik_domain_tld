Cara Pembelian Produk Fisik RAKIFSUL (Wajib Dibaca)

RAKIFSUL March 15, 2024

Berikut ini adalah cara pembelian produk fisik RAKIFSUL yang ada di website ini ( https://fisik.rakifsul.my.id ).

Beli Secara Offline

Semua produk fisik RAKIFSUL hanya dijual secara OFFLINE.

Untuk membeli produk fisik RAKIFSUL, datang langsung ke lokasi saya.

Stok produk terbatas. Jadi, siapa yang duluan beli dia dapat.

Lokasi Saya

Lokasi saya ada di:

Perumahan Pondok Jagung Blok AD.36, Kecamatan Serpong Utara, Tangerang Selatan, Banten, 15326.

Link Google Maps:

Klik Teks Ini

Ketika Sampai di Lokasi

Perlu dicatat bahwa saya tidak membuka toko fisik, jadi, pagar dalam keadaan tertutup.

Biasanya saya siap menerima pengunjung atau menjawab pertanyaan seputar produk setiap hari dari jam 17.00 hingga 19.00 kecuali jika saya lagi ada keperluan.

Jadi, mungkin saja saat Anda datang ke tempat saya, tidak ada orang di sana.

Jika Anda sudah sampai di lokasi saya, Anda bisa panggil nama saya: "Lusfikar! Lusfikar! Lusfikar! Lusfikar! Lusfikar!".

Bilang saja mau beli korek api atau produk lain.

Berteriaklah dengan cukup keras agar terdengar.

Jika cara tersebut tidak berhasil, gunakan fitur chat box yang tombolnya ada di kanan bawah. Katakan saja bahwa Anda sudah ada di lokasi saya.

Tombol chat box hanya muncul saat saya sudah login ke aplikasi chat box.

Pertanyaan Seputar Produk Fisik

Anda bisa chat saya dengan chat box yang tombolnya ada di kanan bawah untuk bertanya tentang produk fisik saya.

Tombol chat box hanya muncul saat saya sudah login ke aplikasi chat box.

Kesimpulan dan Peringatan

  • Baca SELURUH isi halaman ini dengan baik.
  • Semua produk di website ini dijual secara OFFLINE.
  • Saya siap menerima pengunjung atau menjawab pertanyaan seputar produk setiap hari dari jam 17.00 hingga 19.00, kecuali jika saya sedang ada keperluan.
  • Jika Anda sudah sampai lokasi, coba untuk kontak saya baik melalui teriakan maupun chat box. Walaupun begitu, cara tersebut belum tentu berhasil.
  • Tombol chat box hanya muncul saat saya sudah login ke aplikasi chat box.
  • Anda tidak bisa memesan produk, karena siapa duluan beli dia dapat.
  • Janjian ketemuan juga tidak bisa.
  • Saya hanya menerima uang cash sebagai pembayaran produk.
  • Barang yang sudah dibeli tidak bisa ditukar atau dikembalikan.

Terima Kasih

Sekian dan terima kasih.
